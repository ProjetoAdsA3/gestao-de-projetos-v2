
Instruções para o Projeto Gestão de Projetos v2

Este projeto foi desenvolvido em Java com o framework Swing e utiliza o banco de dados MySQL para a persistência de dados.

Requisitos Necessários

Para executar este projeto, é necessário ter instalado:
* **Apache NetBeans IDE 12 ou superior**
* **JDK 8 ou superior**
* **Servidor MySQL** (com MySQL Workbench ou similar)

Configuração do Banco de Dados

Antes de rodar a aplicação, é essencial configurar o banco de dados.

1.  Abra o MySQL Workbench e faça login.
2.  No painel esquerdo, localize a seção "SCHEMAS".
3.  Clique com o botão direito e selecione "Create Schema" para criar um novo banco de dados.
4.  Nomeie o banco de dados como "projects".
5.  Abra o arquivo Untitled.sql
6.  Em uma nova aba de consulta (Query), copie e execute o comando SQL abaixo para criar a tabela necessária:

SELECT * FROM projects;

Como Rodar o Projeto

	1.	No NetBeans, vá em File > Open Project... e selecione a pasta gestaodeprojetosv2. 
	2.	Após a importação do projeto, clique com o botão direito na pasta do projeto e selecione "Clean and Build". 
	3.	Quando a mensagem de BUILD SUCCESS aparecer, clique com o botão direito na pasta do projeto novamente e selecione "Run". 
Obrigado por avaliar o projeto.
